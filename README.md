# Practice Tracker Web App

A Flask-based web application for tracking musician practice time and goals.

## Features
- User registration and login
- Track songs and practice time
- View cumulative progress
- Set monthly goals

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS

## How to Run
1. Install dependencies
2. Run app.py
3. Open in browser using local IP
